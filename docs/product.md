# Echora プロダクト仕様

## 概要

Echora は、発音練習を通じてリスニング力を伸ばす英語学習 Web アプリである。英検 3 級レベルの語彙を対象に、絵と単語を見ながら発音し、AI がリアルタイムでフィードバックを返す。

個人利用とクラス利用の 2 つの形態がある。個人利用ではクラスに所属せず自由に練習できる。クラス利用では、講師が生徒の学習状況を確認できる管理画面が加わる。

## 背景と課題

英検 3 級のリスニングで聞き取れない学習者は多いが、原因は聴覚ではなく発音にあることが多い。

音声知覚のモーター理論（Liberman, 1967）によれば、人は音声を聞くとき、自分の発声運動系と照合して認識する。正しく発音できない音は、聞いても正しく知覚できない。逆に、正しい発音を身につければリスニング力は向上する。

しかし現状の学習環境では、発音練習は軽視されがちである。リスニング教材は「聞く→解く」のサイクルで、発音の正確さに対するフィードバックが存在しない。独学では自分の発音が正しいか判断する手段がなく、塾や英会話スクールでは講師が一人ひとりの発音を確認する時間がない。

Echora は、AI による即時の発音評価と音素レベルのフィードバックでこの課題を解決する。

## ユーザー価値

### 個人利用の学習者

独学では自分の発音が正しいか判断する手段がなく、リスニング教材を繰り返しても聞き取れない音は改善しない。

Echora は音素レベルのフィードバックにより、苦手な音素を特定し、練習の方向性を与える。

### クラスに所属する生徒

従来のリスニング宿題では正解数しか得られず、聞き取れなかった原因は不明なままだった。発音練習は宿題として成立しにくい。

Echora では講師が割り当てた単語セットを自分のペースで練習でき、習得状況と苦手音素を自分でも確認できる。

### 講師

生徒一人ひとりの発音を確認する時間は限られ、リスニングが苦手な原因の特定は難しい。発音指導は経験と勘に依存しがちである。

Echora の管理画面では、生徒ごとの苦手音素や割り当ての習得状況をデータで把握できる。

## 機能概要

### 利用形態

**個人利用**: アカウントを作成し、英検 3 級の語彙から自由に練習する。クラスへの所属は不要。練習履歴、苦手音素、習得状況を確認できる。

**クラス利用**: 講師がクラスを作成し、生徒を招待する。講師は単語セットを課題として配信し、生徒は自分のペースで取り組む。講師は管理画面で各生徒の進捗と苦手音素を確認できる。

### 学習フロー

1. 絵と英単語を表示
2. ネイティブ音声を再生
3. 学習者が発音
4. AI が音素レベルで評価し、フィードバックを表示（正しい音素は緑、誤りは赤、改善ポイントを提示）
5. 任意の回数繰り返す

スコアが 80 以上に達すると「習得済み」となる。回数制限は設けない。

### 練習対象

プロトタイプでは英単語のみを対象とする。英検 3 級レベルの語彙（約 1,300 語）を収録する。

英検 3 級のリスニングには短文も出題されるため、将来的にはフレーズ・短文への拡張を想定する。Speechace API は文レベルの評価にも対応しており、技術的には実現可能である。拡張順序は、単語→フレーズ（2〜3 語）→短文とする。

### 課題の割り当て（クラス利用）

講師はクラスに対して単語セットを課題として割り当てることができる。割り当ては推奨範囲であり、生徒は割り当て外の単語も自由に練習できる。講師は管理画面で割り当てごとの習得状況を確認できる。

### 発音評価

Speechace API の Score Text エンドポイントを使用する。音素・音節・単語レベルのスコアリングに加え、認識された音素（sound_most_like）を返すため、具体的な矯正指示に変換できる。

日本語話者が苦手とする音素（/r/ vs /l/、/θ/ vs /s/、/v/ vs /b/ 等）に基づいた重点フィードバックを提供する。日本語話者の発音に対する API の精度はプロトタイプで検証する。

### 管理画面（クラス利用）

講師向け管理画面では以下の情報を提供する。

- 生徒ごとの練習回数・平均スコアの推移
- 生徒ごとの苦手音素ランキング（混同パターンを含む）
- 割り当てごとの習得状況

個人利用の学習者にも、練習履歴・苦手音素・習得状況を確認できる画面を提供する。

個別の発音録音の再生機能は、プロトタイプのスコープ外とする。

## スコープ

### プロトタイプで実装する機能

- 単語の表示（絵＋テキスト）と音声再生
- マイク録音、Speechace API による発音評価、フィードバック表示
- スコア 80 以上で習得済み判定
- 個人利用：アカウント作成、自由な単語練習、練習履歴・苦手音素の確認
- クラス利用：クラス作成、生徒登録、単語セット割り当て
- 講師向け管理画面：生徒一覧、進捗サマリー、苦手音素、割り当て別習得状況

### プロトタイプのスコープ外

- フレーズ・文レベルの発音評価
- カリキュラムの自動生成・適応的出題
- 保護者向け画面
- 決済・契約管理
- ネイティブアプリ

## ターゲットユーザー

### 個人ユーザー

英検 3 級の受験を控えた中学生〜高校生、または英語をやり直したい社会人。リスニングが苦手で、発音練習の習慣がない層。

### 導入先（クラス利用）

英語塾・英会話スクール。英検対策コースを持つ中小規模の事業者を想定する。

### 購買決定者

塾・スクールの運営者または講師。発音練習によるリスニング力向上に関心があり、生徒の自主学習を仕組み化したいと考えている人。個人ユーザーや保護者からの推薦も導入のきっかけとなりうる。
